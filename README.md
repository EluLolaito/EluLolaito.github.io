# SkyMusic Website

Welcome to the SkyMusic website! This project offers a platform for sharing music sheets, blogs, and small projects. 

## Project Overview

SkyMusic is designed to cater to various needs:
- **Music Sheets**: An interactive table displaying music sheets with detailed information.
- **Blog**: A section for writing and sharing blogs, investigations, and updates.
- **Projects**: Showcase and share small projects.
